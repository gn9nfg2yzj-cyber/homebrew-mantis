# Homebrew Tap for Mantis

Homebrew formulae for [Mantis Security Framework](https://github.com/gn9nfg2yzj-cyber/mantis).

## Installation

```bash
brew install gn9nfg2yzj-cyber/mantis/mantis
```

Or:

```bash
brew tap gn9nfg2yzj-cyber/mantis
brew install mantis
```

## Usage

```bash
mantis           # Interactive mode
mantis status    # Check security status
mantis --help    # Show help
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| mantis | Unified security control for macOS network tools |
